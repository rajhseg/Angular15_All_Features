# Angular15_All_Features
This repo consists of samples for all angular 15 features including custom if else, switch, for directives
Features:
1. Custom Directives like custom if else, custom for loop, custom switch case
2. Events in Components
3. CanActivate, CanDeActivate, CanLoad, CanResolve example
4. PreLoading Startegy
5. Routing Moudle with lazy loading enabled
6. Retry Interceptors
7. Language Internationlization culture change
8. Providers in Module with UseFactory, UseValue, UseClass, UseExisting, Deps sample
